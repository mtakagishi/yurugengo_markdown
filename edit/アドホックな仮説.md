# アドホックな仮説


後付けの追加説明。その場しのぎの仮説



Tip


* 地動説、フロギストン説などがクリシェ
* 詳しくは「教養悪口本」
* 「た」第4回で登場する「た」の解釈に対し「それはアドホックな仮説なのでは」

# シネクドキ


-   ある事柄の代表的な事例ひとつで全体の意味を持つような言語表現
-   ゆる言語学ラジオでは、 `サメの夏`
    という `ミーム` が生まれた。



Tip


\* 「人はパンのみに生くるにあらず」の「パン」 \*
「たほいや」という語彙を当てるゲームにおける「たほいや」


**関連ワード**

-   `サメの夏`
-   `ミーム`

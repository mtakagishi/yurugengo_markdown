# ツーペン


-   堀元氏考案のじゃんけんの改良版
-   ツーッペン、とタメのある独特なリズムが特徴

**ルール**

-   言い出しっぺ、または上座が親
-   グーとチョキだけを使う
-   二人の場合、同じなら親の勝ち
-   三人以上の場合、少数派の勝ち

**関連ワード**

-   `ぐちーち`

# ミーム


-   あったら便利な言葉や分類。ウチワ言葉



Tip


* リチャード・ドーキンスによる造語 
* ゆる言語学ラジオでは、ミームの自然発生が多い。 
* ミーム提案が活発でミーム提案委員会が複数回放送された。


**example**

-   `ソシュールみ`
-   `エターナルエンプティ`
-   `奴隷`
-   `サメの夏`
-   `高村光太郎マインド`
-   `オナマス期`

**関連ワード**

-   `コモンセンス`
